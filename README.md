# Housie Game - Alexa Skill

## Overview
Housie Game is an engaging bingo-style game designed for any number of players. Each participant receives a unique ticket featuring numbers from 1 to 100. As Alexa announces the numbers, players mark them off on their tickets. The winner is the first person to complete their entire ticket by marking all the called numbers.

## Features
- **Multiplayer Support**: Playable by any number of persons
- **Dynamic Content**: The skill generates dynamic game content for each session
- **Ticket Verification**: Check your ticket against announced numbers
- **Voice-Activated Gameplay**: Fully controlled through Alexa voice commands

## How to Use
1. **Launch the Skill**: Say "Alexa, open Housie Game" to start the skill.
2. **Verify Your Ticket**: Say "Alexa, verify the ticket" to check your ticket against the numbers prompted by Alexa.

## Skill Details
- **Invocation Name**: housie game
- **Dynamic Content**: Yes, the skill includes dynamic content for varied gameplay experiences.
